THE ROADMAP TO WEB DEVELOPMENT

Responsive design with HTML, CSS and JavaScript

Foundations of Web Development: HTML and CSS basics, responsive web design using Flexbox and Bootstrap, and JS basics, together with the core concepts of object-oriented programming, DOM manipulation, and version control.

The BACK-END

Try your hand at NodeJS, ExpressJS and MongoDB

Learn how the model-view-controller design pattern works and how to create a server using NodeJS, ExpressJS and Handlebars. Also learn about MongoDB, a non-relational database, as well as Mongoose, an object-data-modeling library. Also learn how to make HTTP requests with Axios, integrate APIs into your application and deploy them to production.

The FRONT-END

Master one of the most widely used front-end frameworks: ReactJS. Learn concepts such as single page app (SPA) and component-based frameworks
